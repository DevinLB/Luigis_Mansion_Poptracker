# Luigi's Mansion PopTracker 0.1.0

This is the PopTracker pack for Luigi's Mansion Archipelago randomizer.

## Installation

Download the lmpt.zip file and install the pack to your PopTracker. On the PopTracker itself, navigate to the folder in the top left and select the version of this tracker that you installed.
